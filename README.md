# rust-the-complete-developer-guide-udmey

#### install rustup
```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

-- verify

rustup update

cargo --version
```



#### new project

```
cargo new deck
```

#### running

```
cargo run

-- or run quitely

cargo run -q
```

#### add a crate

```
cargo add rand
```
